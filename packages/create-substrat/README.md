# create-substrat

Reserves `npm create substrat` for the Substrat initializer.

**The initializer is not released yet.** This package prints a pointer to the docs and
exits. It does not scaffold anything, because a stub that pretended to would be worse than
nothing.

To start a vertical today, follow the [getting started guide](https://substrat.ahlstrand.es/guide/getting-started).
The packages are published and usable:

```sh
pnpm add @substrat-run/kernel @substrat-run/contracts @substrat-run/adapter-sqlite zod
```

## Note on the name

`substrat` on npm is an **unrelated** package — an HTML5 build system published in 2013.
Substrat's own packages are all under the [`@substrat-run`](https://www.npmjs.com/org/substrat-run)
scope. Never `npm install substrat`.
