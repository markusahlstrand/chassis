Archivo (800 for display, tight tracking) + IBM Plex Mono for every time, price and countdown — the transactional data reads as data. Ink #14171A on warm off-white; one accent: court-lime #D7F34F, always paired with ink text and never the only carrier of state (labels + shape everywhere). Urgency amber #C2510F reserved for the hold countdown. Skill scale 0–7. SEK, 24h, km. Slot chips carry fit dots (▪=60 ▪▪=90 ▪▪▪=120) so a start time never dead-ends into an unavailable duration.
